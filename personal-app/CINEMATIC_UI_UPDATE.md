## ✨ 주요 개선 사항

### 1️⃣ **와이드스크린 영화적 레이아웃**

#### 레터박스 효과
```typescript
// 상단/하단 검은 띠로 시네마틱 느낌 강화
상단 레터박스: 그라데이션 (진함 → 투명)
하단 레터박스: 그라데이션 (진함 → 투명)
높이: 전체의 8%
```

**효과**:
- 영화 스크린 같은 몰입감
- 콘텐츠 영역 집중도 향상
- 프로페셔널한 느낌

### 2️⃣ **챕터별 그라데이션 배경**

| 챕터 | 색상 컨셉 | 그라데이션 |
|------|----------|-----------|
| **기** (시작) | 새벽의 푸른 빛 | #DBEAFE → #BFDBFE → #93C5FD |
| **승** (성장) | 햇살의 초록빛 | #DCFCE7 → #BBF7D0 → #86EFAC |
| **전** (긴장) | 석양의 주황빛 | #FFEDD5 → #FED7AA → #FDBA74 |
| **결** (완성) | 밤하늘의 보라빛 | #F3E8FF → #E9D5FF → #D8B4FE |

**개선 효과**:
- 단색 → 3색 그라데이션으로 깊이감 추가
- 135도 대각선 방향으로 역동성 표현
- 각 챕터의 분위기를 색상으로 시각화

### 3️⃣ **영화 클랩보드 스타일 장면 번호**

```
┌─────────────────────────────────────────┐
│ SCENE [02] · 승 (전개)                  │
└─────────────────────────────────────────┘
```

**디자인 특징**:
- 대문자 "SCENE" 레이블
- 장면 번호 하이라이트 박스
- 챕터 정보와 함께 표시
- 그라데이션 배경 + 그림자 효과

### 4️⃣ **영화 타이틀 스타일 제목**

#### 제목 슬라이드
- 크기: **56px** (기존 52px)
- 굵기: **900** (Extra Bold)
- 자간: **-0.03em** (타이트)
- 그림자: 미세한 텍스트 그림자
- 구분선: 그라데이션 하단 라인

#### 장면 제목
- 크기: **36px** (기존 32px)
- 굵기: **800** (Heavy)
- 자간: **-0.02em**
- 그림자: 입체감 추가

### 5️⃣ **영화 자막 스타일 텍스트**

```
┌─────────────────────────────────────┐
│ 🎬 [검은 배경 박스]                  │
│    흰색 텍스트로 영화 자막 표현      │
└─────────────────────────────────────┘
```

**특징**:
- 배경: `rgba(0,0,0,0.85)` (반투명 검정)
- 텍스트: 흰색 (#FFFFFF)
- 패딩: 10px 18px
- 둥근 모서리 + 그림자 효과
- 영화 자막과 동일한 느낌

### 6️⃣ **필름 프레임 스타일 슬라이드 번호**

```
━━━━━━ 01
```

**디자인**:
- 모노스페이스 폰트 (필름 코드 느낌)
- 2자리 제로패딩 (01, 02, 03...)
- 그라데이션 라인과 함께 표시
- 챕터 강조색으로 표현

## 🔧 구현 세부사항

### 수정된 파일

#### 1. `src/lib/slideExport.ts`
```typescript
// SlideData 타입 확장
interface SlideData {
  // ... 기존 필드
  bgGradient?: string;      // 신규: 그라데이션 배경
  accentDark?: string;      // 신규: 어두운 강조색
}

// 색상 매핑 개선
const PANEL_COLOR_MAP = {
  ki: {
    bg: '#EFF6FF',
    bgGradient: 'linear-gradient(135deg, #DBEAFE 0%, #BFDBFE 50%, #93C5FD 100%)',
    accent: '#3B82F6',
    accentDark: '#1E40AF'
  },
  // ... 나머지 챕터
}
```

#### 2. `src/components/slide/SlideCard.tsx`
- **레터박스 효과**: 상단/하단 그라데이션 오버레이
- **클랩보드 장면 번호**: 그라데이션 배경 + 하이라이트 박스
- **영화 자막**: 검은 배경 박스 + 흰색 텍스트
- **필름 프레임 번호**: 모노스페이스 폰트 + 그라데이션 라인

### 타이포그래피 스케일

| 요소 | Before | After | 개선 |
|------|--------|-------|------|
| 제목 슬라이드 | 52px / 800 | **56px / 900** | +4px, 더 굵게 |
| 장면 제목 | 32px / 700 | **36px / 800** | +4px, 더 굵게 |
| 본문 | 24px / 400 | **26px / 500** | +2px, 가독성 향상 |
| 자막 | 15px / italic | **15px / normal** | 영화 자막 스타일 |
| 슬라이드 번호 | 15px / 700 | **16px / 800** | 모노스페이스 |

### 색상 데이터 구조 확장

```typescript
// Before
{
  bg: '#EFF6FF',
  accent: '#3B82F6'
}

// After
{
  bg: '#EFF6FF',                    // 폴백용
  bgGradient: 'linear-gradient(...)', // 그라데이션
  accent: '#3B82F6',                // 기본 강조색
  accentDark: '#1E40AF'             // 클랩보드용
}
```
