# 최고 관리자 시스템 로드맵

> 점조직 형태의 스토리 구성 웹학습지 전국 사용 현황 관리 시스템

## 1. 개요

### 1.1 목적
- 전국에 배포된 점조직 시스템의 사용 현황 파악
- 새 버전 출시 시 사용자에게 업그레이드 알림 메일 발송
- 개발자 차원의 통계 및 유지보수 지원

### 1.2 핵심 철학
- **점조직 독립성 유지**: 각 교사 시스템은 완전히 독립 운영
- **자발적 참여**: 통계 보고는 선택사항
- **개인정보 최소화**: 익명화된 데이터만 수집
- **투명성**: 어떤 데이터가 수집되는지 명확히 고지

## 3. 수집 데이터 명세

### 3.1 필수 데이터 (익명)
| 필드 | 설명 | 예시 |
|------|------|------|
| `systemId` | 시스템 고유 ID (UUID) | `a1b2c3d4-...` |
| `version` | 현재 사용 버전 | `1.0.0` |
| `createdAt` | 시스템 생성일 | `2026-01-15` |
| `lastActiveAt` | 마지막 활동일 | `2026-01-17` |
| `studentCount` | 등록 학생 수 | `25` |
| `classCount` | 학급 수 | `3` |

### 3.2 선택 데이터 (동의 시)
| 필드 | 설명 | 예시 |
|------|------|------|
| `schoolCode` | 나이스 학교 코드 | `B100000123` |
| `schoolName` | 학교명 | `서울OO초등학교` |
| `teacherEmail` | 교사 이메일 | `teacher@sen.go.kr` |
| `region` | 교육청 지역 | `서울` |

### 3.3 수집하지 않는 데이터
- ❌ 학생 개인정보 (이름, 번호 등)
- ❌ 학생 작품 내용
- ❌ 교사 PIN
- ❌ 상세 활동 로그

## 4. 구현 계획

### Phase 1: 기반 구축 (2주)

#### 4.1 점조직 시스템 측 수정
```javascript
// Settings.gs에 추가할 함수
function reportUsageStatistics() {
  // 사용자 동의 확인
  if (!getSetting('allowStatisticsReport')) return;

  const stats = {
    systemId: getOrCreateSystemId(),
    version: VERSION,
    studentCount: getStudentCount(),
    classCount: getClassCount(),
    lastActiveAt: new Date().toISOString()
  };

  // 선택 데이터 (동의 시)
  if (getSetting('shareSchoolInfo')) {
    stats.schoolCode = getSetting('schoolCode');
    stats.teacherEmail = Session.getActiveUser().getEmail();
  }

  // 중앙 서버로 전송
  UrlFetchApp.fetch(STATS_API_URL, {
    method: 'POST',
    contentType: 'application/json',
    payload: JSON.stringify(stats)
  });
}
```

#### 4.2 중앙 API 서버
- **옵션 A**: Google Cloud Functions (무료 티어)
- **옵션 B**: Vercel Serverless Functions (무료)

#### 4.3 데이터 저장소
- **주 저장소**: Google Sheets (간단, 무료)

#### 4.6 이메일 알림
```javascript
// 관리자 도구 - 새 버전 알림 발송
function sendUpgradeNotification(targetVersion, newVersion) {
  const outdatedSystems = getSystemsByVersion(targetVersion);

  outdatedSystems.forEach(system => {
    if (system.teacherEmail) {
      GmailApp.sendEmail(
        system.teacherEmail,
        `[스토리 학습지] 새 버전 ${newVersion} 출시 안내`,
        `
안녕하세요, ${system.schoolName || '선생님'}

스토리 구성 웹학습지의 새 버전 ${newVersion}이 출시되었습니다.

현재 사용 버전: ${system.version}
최신 버전: ${newVersion}

업그레이드 방법:
1. GitHub에서 최신 코드 다운로드
2. Apps Script에 붙여넣기
3. 새 버전으로 배포

변경사항: ${getChangelog(newVersion)}

감사합니다.
        `
      );
    }
  });
}
```

### 5.3 대안: 교육청 이메일 검증
```javascript
// 교육청 이메일 도메인 목록
const VALID_EDU_DOMAINS = [
  'sen.go.kr',      // 서울
  'goe.go.kr',      // 경기
  'pen.go.kr',      // 부산
  'ice.go.kr',      // 인천
  'dge.go.kr',      // 대구
  'dje.go.kr',      // 대전
  'gen.go.kr',      // 광주
  'use.go.kr',      // 울산
  'sje.go.kr',      // 세종
  'gwe.go.kr',      // 강원
  'cbe.go.kr',      // 충북
  'cne.go.kr',      // 충남
  'jbe.go.kr',      // 전북
  'jne.go.kr',      // 전남
  'gbe.go.kr',      // 경북
  'gne.go.kr',      // 경남
  'jje.go.kr',      // 제주
];

function isValidTeacherEmail(email) {
  const domain = email.split('@')[1];
  return VALID_EDU_DOMAINS.some(edu => domain.endsWith(edu));
}
```
