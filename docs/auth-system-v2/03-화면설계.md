# 화면 설계 (점조직 모델)

## 개요

점조직 모델에서는 중앙 집중형 대비 화면이 대폭 단순화된다. 교사 코드 입력, 복잡한 매칭 로직 등이 제거된다.

---

## 1. 학생 화면

### 1.1 메인 (로그인) - URL 접속 시

URL 자체가 특정 교사 시스템을 식별하므로, 별도의 교사 코드 입력이 불필요하다.
**교사가 먼저 학생을 등록해야 사용 가능하다.**

```
┌─────────────────────────────────────────┐
│                                         │
│  📚 김영희 선생님의 스토리 학습지       │
│     서울초등학교 4학년 2반              │
│                                         │
│  ─────────────────────────────────────  │
│                                         │
│  이름                                   │
│  ┌───────────────────────────────────┐  │
│  │  홍길동                            │  │
│  └───────────────────────────────────┘  │
│                                         │
│  번호                                   │
│  ┌───────────────────────────────────┐  │
│  │  15                               │  │
│  └───────────────────────────────────┘  │
│                                         │
│  PIN (숫자 6자리)                       │
│  ┌───────────────────────────────────┐  │
│  │  ●●●●●●                           │  │

│                                         │
│           [ 시작하기 🚀 ]               │
│                                         │
└─────────────────────────────────────────┘
```
